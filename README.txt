JAVA by itself can not play .mp3 files.  This project uses MP3SPI to provide the mp3 player.

http://www.javazoom.net/mp3spi/mp3spi.html


The following jars must be on the classpath:

mp3spi1.9.5.jar
jl1.0.1.jar
tritonus_share.jar

The project also requires the java driver for MongoDB:

https://github.com/mongodb/mongo-java-driver/downloads

MongoDb itself may be down loaded from:

http://www.mongodb.org/downloads
